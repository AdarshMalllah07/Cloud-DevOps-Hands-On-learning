# AWS Learning Journey — Core Concepts & Fundamentals

## Overview

Today I focused on understanding the core fundamentals of AWS and cloud computing concepts. Instead of only learning services individually, I studied how AWS infrastructure, networking, security, monitoring, storage, and cloud architecture work together in real-world environments.
