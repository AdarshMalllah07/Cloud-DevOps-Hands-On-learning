\# AWS Learning Summary



Today I studied AWS cloud computing fundamentals and gained a deeper understanding of cloud infrastructure, networking, security, monitoring, storage, and DevOps concepts. I explored how core AWS services like EC2, S3, IAM, VPC, CloudWatch, Lambda, ECS, and CloudTrail work together in real-world cloud environments. :contentReference\[oaicite:0]{index=0}



I learned important infrastructure concepts including:

\- Regions, Availability Zones, and Global Infrastructure

\- EC2 deployment and storage concepts (EBS, EFS, AMI)

\- IAM users, roles, policies, MFA, and access control

\- VPC networking, subnets, route tables, NAT Gateway, Security Groups, and NACLs

\- Monitoring and auditing using CloudWatch, CloudTrail, and AWS Config

\- Docker containers, ECS, EKS, Fargate, and serverless computing concepts 



I also studied advanced operational concepts like:

\- Infrastructure monitoring and alerting

\- Event-driven architecture using EventBridge

\- Auto remediation workflows

\- Cloud security and encryption using KMS

\- Log analysis and troubleshooting

\- Real-world DevOps and cloud operations practices 



This learning helped me understand how scalable, secure, and highly available cloud systems are designed and managed in modern AWS environments.
